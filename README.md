# 🎬 MovieWiki

## 👋 서비스 소개

MovieWiki는 Movie + Wikipedia의 합성어로 영화와 관련된 모든 정보를 문서화하여 공동 편집하고,
문서의 버저닝과 롤백을 지원하는 영화 전문 위키피디아 서비스 입니다.
